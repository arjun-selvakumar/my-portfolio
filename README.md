# My Portfolio

A simple web portfolio built with Vite, vanilla HTML, CSS, and JavaScript.

## Features

- Responsive design
- Clean and modern layout
- Sections for About, Projects, and Contact

## Usage

1. Install dependencies:
   ```
   npm install
   ```

2. Start the development server:
   ```
   npm run dev
   ```

3. Open your browser and visit `http://localhost:5174`

4. To build for production:
   ```
   npm run build
   ```

5. Preview the production build:
   ```
   npm run preview
   ```

## Customization

- Edit `src/main.js` to modify the content
- Update `src/style.css` for styling changes
- Change `index.html` for meta tags or structure

## Troubleshooting

- Ensure Node.js and npm are installed
- If port 5173 is in use, Vite will automatically use the next available port
- For production deployment, upload the `dist` folder contents to your web server
